## Анализ подозрительности пользователя

```mermaid
sequenceDiagram
    participant Frontend
    participant Duely
    participant PostgreSQL
    participant Analyzer

    loop Сбор действий пользователя во время дуэли
        Frontend->>Duely: (1) Список действий пользователя
        Duely->>PostgreSQL: (2) Список действий пользователя
        PostgreSQL-->>Duely: (3) Ок
        Duely-->>Frontend: (4) Ок
    end

    Duely->>Duely: (5) Дуэль закончилась
    Duely->>PostgreSQL: (6) Взять все действия пользователя в дуэли
    PostgreSQL-->>Duely: (7) Все действия пользователя в дуэли
    Duely->>Analyzer: (8) Проанализировать список действий пользователя
    Analyzer-->>Duely: (9) Степень подозрительности
```

- Во время дуэли `Frontend` собирает список действий пользователя и с заданным интервалом передаёт его в `Duely` (1, 4)
- `Duely` сохраняет их в `PostgreSQL` (2, 3)
- После окончания дуэли (5) `Duely` берёт все сохранённые действия пользователя в дуэли из `PostgreSQL` (6, 7)
- Для анализа списка действий пользователя `Duely` делает запрос в `Analyzer` (8) и в ответе получает степень подозрительности (9)
